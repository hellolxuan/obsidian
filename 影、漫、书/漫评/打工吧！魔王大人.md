# 打工吧！魔王大人

一不留神就把第一季给看完了，作为一部喜剧恋爱番的话还不错，节奏一般般，人物也各有特色，而且还是反穿越系的类型。

一开始勇者出场的时候，我还以为这是一部经典的亚萨西男主和傲娇女主的狗粮盛宴，结果半路杀出来一个女二小千，而且是在剧中占大篇幅的重要角色，或者说主动角色，因为很多剧情都是由小千来主动推进的。在剧中小千直接当面表白，是个可爱又害羞的女高中生，同时魔王和勇者之间的关系也时而清晰时而模糊，实在是让我难以捉摸这部剧的恋爱关系，不过较为明显的是非后宫类型。

听说小说的结局是魔王最后和小千在一起了，倒也不是很惊讶，因为按照剧情来讲小千确实更担心魔王，对魔王也更喜欢更上心。但其实要说般配的话，还是勇者更胜一筹吧，毕竟偶尔傲娇、偶尔乖巧的模样确实很加分，而且第一季的最后一集的最后几秒，是魔王和勇者第一次见面并产生缘分的地方，并且重现了借伞和还伞的剧情，最后一幕是勇者开心的转伞，怎么看都是女主角的戏份。

不过勇者因为过去的家人和村庄被魔王杀害和摧毁，这些在剧情上，如果处理的不恰当就很难继续接下来的恋爱剧情，虽然第一季经常出现魔王和勇者被误认为是情侣的情况，但实际的恋爱关系可谓是：毫无进展；反观小千，所有剧情上都能妥善处理，不仅推进了【担心魔王】的剧情，也出现了【担心小千】的剧情，甚至还有【直面告白】的剧情，虽然一般情况下先表白的角色大多最后都成败犬，但这部剧可能会不一样。

总而言之，动漫中应该想表达一种：勇者和小千谁最后成正宫的比例为55开的样子。但我其实觉得至少46开。
